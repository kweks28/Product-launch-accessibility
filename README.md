 5COM2002 - Accessibility and Usability Assignment

Project Overview
University of Hertfordshire assignment focusing on developing an accessible product launch page for "Product X Smartwatch" conforming to WCAG 2.1 Level AA standards.

File Structure
Development Approach
1. Baseline Analysis: Recreated original page from screenshots and conducted initial WAVE testing
2. Systematic Improvement: Progressive accessibility fixes with regular Git commits
3. WCAG AA Compliance: Targeted conformance to Level AA standards
4. Heuristic Evaluation: Usability assessment using Nielsen's principles

Accessibility Features Implemented
- Semantic HTML5 structure with ARIA landmarks
- Keyboard navigation and focus management
- Color contrast compliance (4.5:1 minimum)
- Screen reader compatibility
- Responsive design with accessible touch targets
- Form labels and error handling

WCAG 2.1 AA Conformance
This project aims to meet all Level AA success criteria through systematic testing and improvement.
